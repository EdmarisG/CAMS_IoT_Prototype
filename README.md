# CAMS IoT - Prototipo de Monitoreo de Presion 
 
## Descripcion 
 
Sistema IoT para monitoreo de presion en ventilacion simulada. 
 
## Estructura 
 
- firmware/ - Codigo del ESP32 
- backend/ - API REST Flask 
- dashboard/ - Frontend web 
 
## Archivos Importantes 
 
- .gitignore - Archivos a NO subir a Git 
- .exclude - Archivos a NO incluir en backups 
- firmware/secrets.h.template - Template de credenciales
